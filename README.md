# PANG3 - 사하구청 지도 기반 외근 도우미

외근 업무를 효율적으로 관리하기 위한 지도 기반 시스템입니다.  
위치 기반 기술과 데이터 분석을 활용하여 외근 경로 관리, 업무 기록, 보고서 자동화를 지원합니다.

---

## 👥 팀 소개 (PANG3)

- 김남형 (팀장, 보안)
- 조영준 (클라우드)
- 박기범 (백엔드)
- 모준호 (데이터 분석)

※ 역할은 추후 확정 예정

---

## 🎯 프로젝트 목표

- 외근 업무의 비효율적인 관리 문제 해결
- 지도 기반 위치 추적 및 경로 관리 기능 제공
- 데이터 기반 행정 의사결정 지원
- 안전한 개인정보 및 업무 데이터 관리

## 🧩 역할 분담 (초안)

### 1. ☁️ 클라우드 엔지니어
- Docker 기반 개발 환경 구축
- Aws Cloud 서버 인프라 구성
- CI/CD 파이프라인 구축 (GitHub Actions)
- 서버 모니터링 및 운영 관리

### 2. 🔐 보안 엔지니어
- VPC 기반 네트워크 보안 설계
- JWT 기반 인증 및 권한 관리
- SSL/TLS 통신 및 데이터 암호화
- 로그 관리 및 보안 감사 체계 구축

### 3. ⚙️ 백엔드 개발자
- 위치 기반 데이터 처리 (GIS)
- 이미지 메타데이터(EXIF) 분석
- AI 기반 보고서 생성 기능 구현
- 업무 상태 관리 및 API 개발

### 4. 📊 데이터 분석가
- 공간 데이터 시각화 (Heatmap, Choropleth)
- 경로 최적화 알고리즘 (TSP)
- 통계 분석 및 이상 탐지
- 데이터 파이프라인 구축 (ETL)
  

#######################################

- Server OS: Ubuntu 24.04 LTS (AWS EC2)
- Runtime: Docker / Docker Compose
- Database: PostgreSQL 16 + PostGIS (예정)
- Public IP: 개발 시작하면 매일 아침마다 업데이트 예정 (탄력적 ip 사용 x)
