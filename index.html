배경 & 필요성 02 / 10

왜 이 시스템이 필요한가?

🗂️

아날로그 방식의 비효율성

공무원·영업사원이 현장 조사 후 사무실에 복귀하여 사진·위치도를 HWP에 수동으로 붙여넣는 불필요한 행정력 낭비가 심각합니다.

📊

데이터 활용의 단절

현장 데이터가 비정형 HWP 문서 속 이미지로만 존재하여 통계 분석·시각화 등 데이터 자산으로서 활용이 불가능합니다.

🔗

통합 플랫폼 부재

경로 탐색(네이버 지도)과 보고서 작성(한글)이 분리 운영되어, 공무원·택배·영업 등을 아우르는 범용 통합 솔루션이 없습니다.

🚀

높은 시장성 및 확장 가능성

온나라 지식 업무 프로그램 다운로드 2위가 외근업무도우미(공무원 자체 개발)로, 실수요가 검증된 분야이며 상용화 시 높은 시장성이 기대됩니다.

기능 ① 04 / 10

스마트 위치 관리 & 경로 최적화

📤

일괄 등록

엑셀/CSV 파일 업로드로 대량의 주소지를 DB에 일괄 등록합니다. 반복 수작업을 완전히 제거합니다.

📌

자동 지오코딩

주소 텍스트를 위·경도 좌표로 자동 변환하여 지도에 즉시 매핑합니다. 수동 좌표 입력 불필요.

🔄

경로 최적화 (TSP)

등록된 다수 목적지를 최단 거리·최소 시간으로 방문할 수 있는 최적 순서를 자동 계산합니다.

❌ 수동으로 지도 앱 + 문서 따로 확인
→
✅ 앱 하나에서 목적지 등록 → 최적 경로 즉시 제공
❌ 주소를 일일이 검색하여 좌표 수집
→
✅ CSV 업로드 한 번으로 전체 자동 지오코딩
기능 ② 05 / 10

현장 중심 모바일 기능 Field UX

🔔

도착 알림 시각화

목적지 반경 내 진입 시 마커가 깜빡임·색상 변경으로 반응하여 작업 준비를 직관적으로 유도합니다.

🔄

상태 기반 워크플로우

사진 촬영 후 원터치 상태 변경 (배송전→완료 / 파손확인→보수완료)과 키워드·음성 메모 입력 지원.

⭐ 핵심 기능

📷 EXIF 자동 매칭

현장에서 촬영한 사진의 GPS 메타데이터(EXIF)를 분석하여 지도 위 가장 가까운 포인트에 자동으로 매칭·저장합니다. GPS 오차를 위한 수동 보정 기능도 포함.

📅

시계열 이력 관리

동일 지점의 공사 전→중→후 사진을 날짜별로 관리하고 주제별 지도를 저장·조회합니다.

기능 ③ 06 / 10

AI 기반 보고서 자동화

🤖 LLM 활용

문안 자동 생성

현장에서 입력한 단편적인 메모를 AI(LLM)가 분석하여 정식 보고서 문체로 자동 변환·완성합니다.

💬

입력 예시

"가로등 파손 심각"
"무단투기 다발"

📄

원클릭 문서 출력

위치도(지도 캡처) + 현장 사진 + AI 생성 텍스트를 결합하여 즉시 다운로드.

PDF 출력 Hwpx 출력 Word 출력
⚡

기대 효과

  • 복귀 후 문서 작업 시간 대폭 단축
  • 일관된 보고서 품질 유지
  • 현장→문서 원스톱 처리
기능 ④ 07 / 10

데이터 시각화 대시보드

현장 데이터를 지도 위에서 한눈에 파악하는 인텔리전트 대시보드

🗺️

지도 시각화 레이어

행정 구역(시·도/구·군/읍·면·동) 경계 표시 및 코로플레스(단계구분도) 구현으로 구역별 현황 파악.

🔥

히트맵 & 클러스터링

데이터 밀집도에 따른 히트맵과 마커 클러스터링으로 문제 지역을 직관적으로 식별.

🎯

필터링 & 툴팁

업무 진행률(완료/미완료)에 따른 색상 구분과 직관적인 툴팁으로 실시간 현황 모니터링.

📊 비정형 문서 속에 묻혀있던 현장 데이터가 통계 분석·시각화 가능한 데이터 자산으로 전환됩니다.
